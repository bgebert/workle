WHAT'S NEW && THINGS TO DO
Note: Please direct all bugs/issue/enhancements/questions to Brad Gebert

New fixes/features as of 2022.03.07

  • The gameboard is now always centered over the keyboard depending on the screen width
  • Increased the size of the keyboard
  • Only 1 unique game per day, so everyone plays the same game on the same day.
  • User's can play past games after they play today's game.
  • Based on the previous 2 bullet points, the game selection order of precendence is...
    • 1. Any game requested via URL parameter (sharing with a friend)
    • 2. Today's game
    • 3. An previous day's game that had NOT been played by the user. Repeat as desired
    • 4. Once all previous games have been played, the user would be presented with today's game again.
  • Show a clue when present in the scheduler. The clue button becomes visible when the user only has 1 try remaining.
  • When validating if a user's input is a word, WORKLE now compares against the word list dictionary AND the scheduler.
  • Created a scheduler file that contains the following:
  • Word of the day is now pulled from the scheduler.
  • Adjust gameboard size, word list comparison, and number of tries based off the word of the day
  • Present a clue, if present in the scheduler file AND the user is down to 1 guess remaining
  • Present a shout out to Alex and the Research team for all of their amazing help and ideas.

HOW TO PLAY
  • Guess the correct word in 6 tries.
  • Each guess is submitted by pressing the 'ENTER' button.
  • The tiles will change color to show how close your guess was to the answer.
Examples:

R A I S E
The letter R is in the answer and in the correct place.

V A L U E
The letter L is in the answer but in the wrong place.

U N I T Y
The letter Y is in not the answer.
